Formatação de texto

Negrito:

Voce pode deixer qualquer texto em Negrito usando a tag strong

Italico:

Voce pode deixar um texto em Italico usando a tag em

Marca texto:

Para usar o Marca Texto, usamos a tag mark

Texto pequeno:

Um Texto pequeno se faz com a tag small

Texto deletado:

Para fazer um Texto deletado use a tag del

Texto inserido:

O Texto inserido se faz com a tag ins

Texto superior:

O Texto superior é feito com a tag sup

Texto inferior:

Já o Texto inferior é feito com a tag sub

Codigos:

Para colocar codigos no HTML5, use a tag code

Ex:

num = int(input('Digite um numero: ')) if num % 2 == 0: print(f'O numero {num} é PAR') else: print(f'O numero {num} é IMPAR')

Para deixar tudo bonitinho use a tag pre

Ex:


        num = int(input('Digite um numero: '))
        if num % 2 == 0:
            print(f'O numero {num} é PAR')
        else:
            print(f'O numero {num} é IMPAR')

Citações:

Para fazer Citações use a tag q
